# Análise de Cancelamento de Clientes (Churn Analysis)

## 📊 Sobre o projeto

Este projeto realiza uma análise exploratória de dados com o objetivo de identificar padrões e fatores associados ao cancelamento de clientes (churn).

A análise busca compreender quais características ou comportamentos estão mais relacionados à saída de clientes, gerando insights que podem auxiliar empresas na criação de estratégias de retenção.

## 🎯 Objetivo

Identificar variáveis e padrões que influenciam o cancelamento de clientes, utilizando técnicas de análise de dados e visualização.

## 🛠 Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

## 🔎 Etapas da análise

1. Importação e exploração da base de dados
2. Limpeza e tratamento de dados
3. Análise exploratória (EDA)
4. Visualização de padrões de cancelamento
5. Identificação de variáveis relevantes

## 📈 Exemplos de análises realizadas

- Distribuição de clientes que cancelaram vs. permaneceram
- Relação entre tempo de contrato e cancelamento
- Impacto de serviços contratados no churn
- Análise de variáveis demográficas e comportamentais

## 💡 Principais insights

A análise permite identificar fatores que aumentam a probabilidade de cancelamento, fornecendo informações que podem ser utilizadas para:

- reduzir churn
- melhorar estratégias de retenção
- apoiar decisões de negócio baseadas em dados
